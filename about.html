Link Flagger

The link flagger will scan every link tag in brightspace and see if it passes certain tests

Differently colored links represent different issues:

  • Blue: No Title attribute found
  • Yellow: Link does not open in a new tab
  • Red: The filepath doesn't have the words 'Course Files' or 'Content Files'.

HTML Fixer

The HTML Fixer was built to correct poorly marked up content pages on the transition from I-Learn 2.0. It will scan html in content pages, and fix many common issues.

The tool ads a button to the edit html frame, which execute the following when clicked:

  • Changing <b> to <strong>
  • Changing <i> to <em>
  • Removing empty tags
